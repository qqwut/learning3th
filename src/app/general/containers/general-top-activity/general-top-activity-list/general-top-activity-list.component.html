คอร์ส Seleniumm + Python เทสหน้าเว็บแอพ

คอร์สสำหรับการเทสหน้าเว็บแอพพลิเคชั่นด้วยการใช้ Selenium มาช่วยในการจัดการกับบาง element ที่บางที robotframework ไม่สามารถทำได้ คอร์สนี้รวมไปถึงการใช้งาน Python ขั้นพื้นฐานเพื่อให้ไปต่อยอดกับงานจริงได้ในอนาคต *สำหรับผู้ที่ทำเทสหน้าแอพพลิเคชั่นและผู้เริ่มต้นศึกษา ไปจนถึงขั้นใช้งานได้จริง

คอร์สทดสอบ Web Application ระดับกลาง ด้วย Robot Framework และ Selenium web driver

เรียนรู้ การทดสอบ Web Application ระดับกลาง ด้วย Robot Framework และ Selenium web driver โดยคอร์สนี้จะเป็นคอร์สระดับกลาง เพื่อช่วยให้ผู้เรียนเข้าใจการจัดการ Test Script ได้อย่างมีประสิทธิภาพมากขึ้น
- รู้จักเทคนิคการจัดเก็บ Test Script ให้ปลอดภัย
- การเขียนและดูแล Test Script ได้อย่างมีประสิทธีภาพ ด้วย Resource file และ Page Object
